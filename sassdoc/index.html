    flex-grid-row

    @mixin flex-grid-row($behavior: null, $width: $grid-row-width, $columns: null, $base: true, $gutter: $grid-column-gutter) { ... }

    Description

    Creates a container for a flex grid row.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $behavior

    Modifications to the default grid styles. nest indicates the row will be placed inside another row. collapse indicates that the columns inside this row will not have padding. nest collapse combines both behaviors.

    		Keyword or Listnull
    $width

    Maximum width of the row.

    		Number$grid-row-width
    $columns

    Number of columns to use for this row. If set to null (the default), the global column count will be used.

    		Numbernull
    $base

    Set to false to prevent basic styles from being output. Useful if you're calling this mixin on the same element twice, as it prevents duplicate CSS output.

    		Booleantrue
    $gutter

    Gutter to use when inverting margins, in case the row is nested.

    		Number$grid-column-gutter

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Requires

    • [mixin] grid-row-nest
    • [mixin] grid-context

    flex-grid-column

    @function flex-grid-column($columns: null) { ... }

    Description

    Calculates the flex property for a flex grid column. It accepts all of the same values as the basic grid-column() function, along with two extras:

    • null (the default) will make the column expand to fill space.
    • shrink will make the column contract, so it only takes up the horizontal space it needs.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $columns

    Width of the column.

    		Mixednull

    grid-column

    @function grid-column($columns) { ... }

    Description

    Calculates the width of a column based on a number of factors.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $columns

    Width of the column. Accepts multiple values:

    • A percentage value will make the column that exact size.
    • A single digit will make the column span that number of columns wide, taking into account the column count of the parent row.
    • A string of the format "x of y" will make a column that is x columns wide, assuming y total columns for the parent.
    		Number or List—none

    Returns

    Number —

    A calculated percentage value.

    Throws

    • Wrong syntax for grid-column(). Use the format

    • Wrong syntax for grid-column(). Use a number, decimal, percentage, or

    • [variable] grid-column-count
